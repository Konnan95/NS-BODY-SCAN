1. Какие данные собираются

  • Регистрационные данные: имя, email, возраст, рост, вес
  • Биометрические данные: фото и видео для анализа осанки и техники упражнений
  • Данные активности: шаги, сон, питание (вводимые пользователем)
  • Технические данные: IP-адрес, тип устройства, браузер

2. Как используются данные

  • Для анализа осанки и состава тела
  • Для генерации персонализированных планов тренировок и питания
  • Для отслеживания прогресса
  • Для связи с тренером
  • Для улучшения сервиса (анонимная статистика)

3. Хранение данных

  • Все данные хранятся в защищённой базе данных PostgreSQL
  • Пароли хешируются (никто не может их прочитать)
  • Фото и видео хранятся на сервере с ограниченным доступом
  • Данные не передаются третьим лицам без вашего согласия

4. Ваши права

  • Вы можете запросить удаление всех своих данных
  • Вы можете экспортировать свои данные (CSV/PDF)
  • Вы можете отказаться от обработки биометрических данных
