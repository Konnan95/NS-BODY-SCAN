## 📱 О проекте

NS Body Scan — это полноценное веб-приложение для AI-анализа осанки по фото, генерации персонализированных планов тренировок и питания, а также отслеживания прогресса.

### 🚀 Основные возможности

| Функция | Описание |
|---------|----------|
| **📸 AI-анализ осанки** | MediaPipe (33 ключевые точки), 3 ракурса, визуализация |
| **🧬 Анализ состава тела** | 3 формулы (US Navy, YMCA, BMI) + ML модель |
| **🎥 Анализ техники упражнений** | Видео с обратной связью (приседания, отжимания, выпады и др.) |
| **🤖 AI-чат-бот** | GigaChat — отвечает на вопросы о тренировках и питании |
| **🔊 Голосовые рекомендации** | TTS (Text-to-Speech) для озвучивания результатов |
| **📚 Библиотека упражнений** | 1500+ упражнений с GIF-демонстрациями, поиск, фильтры |
| **📊 Графики прогресса** | Chart.js — динамика веса, % жира, шагов, сна |
| **🥗 Дневник питания** | FatSecret API — поиск продуктов, КБЖУ, история |
| **📅 Календарь тренировок** | Планирование, отметка выполнения, статистика |
| **👨‍🏫 Кабинет тренера** | Клиенты, чат, редактирование программ и планов питания |
| **📋 Админ-панель** | Статистика, управление пользователями, графики активности |
| **🎁 Промокоды и скидки** | Генерация и применение промокодов при подписке |
| **🎥 Видео-звонки** | Jitsi Meet — бесплатные звонки тренер-клиент |
| **🌙 Тёмная тема** | Переключение светлой/тёмной темы с сохранением выбора |
| **🏆 Достижения** | Геймификация — награды за активность |
