\# 🧠 NS Body Scan



\*\*AI-платформа для анализа осанки и фитнеса\*\*



\---



\## 📱 О проекте



NS Body Scan — это полноценное веб-приложение для анализа осанки по фото, генерации персонализированных планов тренировок и питания с помощью AI.



\### 🚀 Основные возможности



\- \*\*📸 AI-анализ осанки\*\* (MediaPipe, 33 ключевые точки)

\- \*\*🧬 Анализ состава тела\*\* (3 формулы + ML)

\- \*\*🎥 Анализ техники упражнений\*\* (видео с обратной связью)

\- \*\*🤖 AI-чат-бот\*\* (GigaChat)

\- \*\*🔊 Голосовые рекомендации\*\* (TTS)

\- \*\*📚 Библиотека упражнений\*\* (1500+ GIF)

\- \*\*📊 Графики прогресса\*\* (Chart.js)

\- \*\*🥗 Дневник питания\*\* (FatSecret API)

\- \*\*👨‍🏫 Кабинет тренера\*\* (клиенты, чат, редактирование программ)

\- \*\*📋 Админ-панель\*\* (статистика, управление пользователями)

\- \*\*🎁 Промокоды и скидки\*\*

\- \*\*🎥 Видео-звонки\*\* (Jitsi Meet)
